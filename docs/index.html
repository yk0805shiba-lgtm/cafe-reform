新宿エリア イベントカレンダー（Market Intelligence）

公開アクセスについての注意:
このページは公開されています。個人情報・秘密情報は含まれていません。
掲載されているのは外部の公開イベント情報のみです。店舗名・スタッフ個人名・売上データ等の内部情報は一切含まれません。

ICS ファイル一覧（カレンダー購読用）

  • all.ics 全イベント
  • cafe.ics カフェ営業向けイベント
  • delivery.ics デリバリー向けイベント
  • cafe_01_cafe.ics 店舗 cafe_01（カフェ）
  • delivery_01_delivery.ics 店舗 delivery_01（デリバリー）

Google カレンダーへの追加方法

  1. Google カレンダー を開く
  2. 左サイドバーの「他のカレンダー」横の「+」→「URL で追加」をクリック
  3. 上記 ICS ファイルの URL を貼り付けて「カレンダーを追加」

ICS の購読 URL は GitHub Pages を有効化した後、以下の形式になります（YOUR_GITHUB_USERNAME を書き換えてください）:

  • HTTPS: https://YOUR_GITHUB_USERNAME.github.io/cafe-reform/market-intelligence/events/all.ics
  • webcal: webcal://YOUR_GITHUB_USERNAME.github.io/cafe-reform/market-intelligence/events/all.ics

Apple カレンダーへの追加方法

  1. Apple カレンダーアプリを開き、「ファイル」→「新規カレンダーの購読」を選択
  2. 上記 ICS ファイルの URL を入力
  3. 更新頻度は「毎日」を推奨
