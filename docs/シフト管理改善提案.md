# シフト管理改善提案書

## 現状の問題

### 根本原因
シフトの決定が全て手動で行われており、人的ミスが頻発している。

### 問題の連鎖
- 誰が何度出勤し、どれくらい働けるか、モチベーションがあるかを区別できていない
- 開け・締め作業の知識と経験がある人材の把握ができていない
- シフトを通してしか観察できていないため、スタッフの理解度・吸収力が不明
- アルバイトの意見を聞ききれていない
- 接触頻度が少なく、問題が直前まで発覚しない
- 穴を作らないことが優先となり、問題が先延ばしされている

## 改善案

### 1. シフト提出時の定期連絡
- シフト提出のタイミングで必ず個別に連絡を入れる
- 開け作業が一人で可能か、自信があるかを確認する
- できない場合は早めに対応し、問題を先延ばしにしない

### 2. 保険（代打）制度の導入
- シフト提出が遅い場合、代打を立てて「もう出れない」と認識させる
- 少しでも働きたければ提出を早くするインセンティブになる
- 早めの相談であれば自分（インターン）が穴埋めに入ることも可能

### 3. モチベーションの可視化
- 積極的にシフトを出すスタッフを把握し、貢献度を差別化する
- 店舗に貢献しているスタッフがわかりやすくなり、シフトが組みやすくなる

## 期待される効果
- シフトの穴埋め対応が早くなる
- やる気のあるスタッフが報われる仕組みができる
- 直前のシフト募集・変更が減る
- スタッフのスキル・モチベーションの可視化が進む

## 現状のヒアリング結果
- 昼・夜メインで働きたくても働けないスタッフが多い
- 少しでも多く働きたいというスタッフが多数いる
- この層を活かせる仕組みがまだできていない
